# 直接检测方法实验设计

## 实验目标

验证直接检测方法（CAM + 图像特征 → 直接预测框）是否比阈值方法更有效。

## 实验设计

### 实验组：直接检测方法
- **模型**: DirectDetectionDetector
- **检测头**: DirectDetectionHead（融合CAM和图像特征）
- **损失函数**: DirectDetectionLoss（基于IoU分配正样本）
- **训练epoch**: 50个epoch（快速验证）

### 对比基线：阈值方法
- **模型**: SimpleSurgeryCAMDetector（当前最佳：实验2.1c，损失2.6004）
- **方法**: 固定阈值峰值检测

## 评估指标

1. **训练损失**: L1, GIoU, 置信度, CAM损失
2. **正样本比例**: 应该>0（有正样本才能训练）
3. **收敛速度**: 损失下降速度
4. **最终性能**: mAP@0.5（训练后评估）

## 成功标准

1. ✅ 正样本比例 > 0（有正样本）
2. ✅ 损失持续下降
3. ✅ 最终损失 < 2.5（优于基线2.6004）
4. ✅ 训练稳定（无震荡）

## 实验步骤

1. 运行直接检测训练（50个epoch）
2. 监控训练过程
3. 评估训练后的模型
4. 对比基线方法
